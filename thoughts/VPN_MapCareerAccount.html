Conecting to Purdue VPN from a Linux system and mapping career account

Connecting to Purdue VPN

  1. Install network-manager-vpnc
    (If on Ubunutu, can install from application store)

  2. Click on:

    1. The network icon
    2. VPN connections
    3. Configure VPN

  3. Select Add

    Select Cisco Compatible VPN (vpnc)

  4. Name the network as convenient

  5. Under VPN tab

    1. Set Gateway as: webvpn.purdue.edu
    2. Set userName as your career account name
    3. Type in userPassword if you want to save it
    4. Set GroupName PurdueUser
    5. Set GroupPassword as jtgkld1990
    6. Save and close all related windows

  6. Click:

    1. The network icon
    2. VPN connections
    3. Select the VPN network you just configured
    4. If you did not provide UserPassword, you will be promped to enter it here

Mapping to Purdue career account drive

  1. Open file browser

  2. Click on:

    1. File [ALT + F]
    2. Connect to Server... [S]

  3. In the pop-up menu

    1. In server address fill: smb://myhome.itap.purdue.edu/myhome/USERNAME
    2. Where USERNAME is your career account log-in

  4. In the new pop-up menu

    1. Fill in uername as career account login
    2. In Domain fill in onepurdue
    3. Then enter your password
    4. Press connect

  5. In your file browser, under the Network segment you can view the drive files
