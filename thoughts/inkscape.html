My day with Inkscape

Before I list out key tools I used today, the following were really helpful resources:
  • Chris' tutorial
  • YouTube tutorial
  • The community online is really awesome too!

These tutorials by themselves majorly covered everything I used for my logo. I wanted to list the tools I over-abused during my design process.

On the menu on the left, the rectangle tool was used a lot by me. The best part was the small circle that allowed rounding the edges.
I found the pen tool for straight lines and bezier lines very useful too.
The text tool is an essential that is hard to miss out on.
The Edit Path by Nodes was helpful in fixing mistakes.

With regards to sub menus, the Fill and stroke menu was used a lot to color objects and create gradients (majorly for shadows).
The layers menu was essential is dividing logo into parts into layers allowing modification of parts at a time.
The grid option was of great help in aligning elements on the logo.

Some things to note (for me) were:
  1. Convert text to objectToPath form
  2. Shadow can be implemented as a layer under object that requires one
  3. There is always a nifty hack (like the one i discoved on shadows)
