Calculator GUI



Description:

Implementation of a basic calculator that carries out fundamental operations such as addition, subtraction, division and multiplication following mathematical precedence of calculation in addition to basic display formatting.
Program was written in python using the QT framework and pySide.
Link to github repository here.

My learning experience

  • Designing graphic user interfaces in QT designer.
  • Using PySide tool to allow programming in python.
  • Event driven programming in python.

Some challenges faced

Challenges faced

  • Incorporate design that allows objects to be placed neatly and clearly.
  • Linking QT design to python using PySide.
  • Designing and Implementing an algorithm to solve mathematic expressions following correct precedence of operators while looking for potential errors in the program.
  • Formatting the output to print required decimal points and toggle between comma (,) in the output.
