Frequently Asked Questions

Because... information transparency


  • When will (insert language) be added ?

    Popular languages that can fit into this framework are constantly being added one after another.
    Having just one developer makes things come out a little slowly since focus is on quality and testing takes time.
    I would appreciate it if, in the feedback section, you would add your language.

  • Will a spell check feature for English be added ?

    Most definitely it will. Unfortunately, it is a hard feature to programme and so it will take time to test and release.

  • Should I worry about this keyboard stealing information written by me ?

    This is a great and very important question.
    NO! Apurva Keyboard does not save, transmit or re-use any thing you type.
    Another great reason to use Apurva Keyboard!

  • What permissions does Apurva Keyboard use, and why ?
    Android permissions used are:
    • VIBRATE (To vibrate the phone on keypress)
    • INTERNET (To send trivial data such as language used and for how long)
    • WAKE LOCK (To keep phone on when using internet)
    • ACCESS NETWORK STATE (To save battery when using internet)

    Yup! That's it. Simple is better, no ?
